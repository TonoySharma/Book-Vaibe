Book Vibe is a modern book tracking web application where users can explore books, mark them as read, and save favorites to their wishlist. Built with React and Tailwind CSS, it provides a smooth and responsive user experience.

🚀 Features
📖 Browse and view book details
❤️ Add books to Read List
🔖 Save books to Wishlist
🔍 Sorting system (by Pages & Rating)
📱 Fully responsive design (mobile, tablet, desktop)
🎨 Clean and modern UI with Tailwind CSS
⚡ Fast navigation using React Router

🛠️ Tech Stack
React.js
React Router
Context API
Tailwind CSS
React Icons
